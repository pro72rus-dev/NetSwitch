## Возможности

- **Мгновенное переключение** — отключение и включение всех сетевых адаптеров по горячей клавише
- **Системный трей** — работает в фоне, не занимает место на панели задач
- **Оверлеи поверх игр** — уведомления отображаются поверх полноэкранных приложений
- **Тёмный интерфейс** — минималистичный GUI с настройкой горячей клавиши
- **Двуязычность** — интерфейс и уведомления на English / Русском
- **Русская раскладка** — автоматическое распознавание клавиш с русской клавиатуры
- **Автоустановка** — при первом запуске устанавливается в `%LOCALAPPDATA%\NetSwitch`
- **Самообновление** — проверка обновлений при запуске
- **Деинсталляция** — `NetSwitch.exe --uninstall`

## Горячая клавиша

| Комбинация | Описание |
|------------|----------|
| `Ctrl + End` | Вкл/Выкл интернет (по умолчанию) |
| Любая комбинация через GUI | Нажмите **Change** и клавиши |

Поддерживаются: `Ctrl`, `Alt`, `Shift`, `Win` + любая клавиша, `F1`-`F24`, `End`, `Home`, `Delete` и др.

## Установка

### Автоматическая (рекомендуется)

1. Скачайте [`NetSwitch.exe`](https://github.com/pro72rus-dev/NetSwitch/releases/latest/download/NetSwitch.exe)
2. Запустите — программа установится автоматически
3. Готово! Горячая клавиша `Ctrl + End`

### Ручная

```bash
git clone https://github.com/pro72rus-dev/NetSwitch.git
cd NetSwitch/internet-toggle
pip install -r requirements.txt
python main.py
```

## Сборка

Требуется: Python 3.14+, PyInstaller

```bash
pip install pyinstaller
.\build.ps1
```

Готовый exe будет в `release/NetSwitch.exe`

## Структура проекта

```
internet-toggle/
├── main.py          # Точка входа, горячие клавии, трей, логика
├── gui.py           # Тёмный GUI (tkinter)
├── toggle.py        # Управление сетевыми адаптерами (netsh)
├── notifier.py      # Оверлей-уведомления поверх игр
├── strings.py       # Локализация (EN/RU)
├── build.ps1        # Скрипт сборки
├── requirements.txt # Зависимости
└── start.bat        # Запуск без сборки
```

## Использование

| Действие | Как |
|----------|-----|
| Вкл/Выкл интернет | `Ctrl + End` (или настроенная комбинация) |
| Открыть окно | Двойной клик по иконке в трее |
| Скрыть в трей | Кнопка `─` в заголовке окна |
| Сменить горячую клавишу | **Change** → нажать комбинацию → **Confirm** |
| Сменить язык | Кнопка `EN`/`RU` в заголовке |
| Деинсталляция | `NetSwitch.exe --uninstall` |

## Требования

- Windows 10/11
- Python 3.11+ (для запуска из исходников)
- Права администратора (для управления адаптерами)
