# NetSwitch

**Toggle all internet adapters with a single hotkey**

## What it does

NetSwitch instantly disables/enables all network adapters via a hotkey. Runs in the system tray, notifications appear over fullscreen games.

## Installation

1. Download [**NetSwitch.exe**](https://github.com/pro72rus-dev/NetSwitch/releases/latest/download/NetSwitch.exe)
2. Run the file — installation is automatic
3. Done! Press `Ctrl + End` to toggle internet

## Usage

| Action | How |
|--------|-----|
| Toggle internet | `Ctrl + End` |
| Open window | Double-click the tray icon |
| Minimize to tray | `─` button in the title bar |
| Change hotkey | **Change** → press keys → **Confirm** |
| Switch language | `EN`/`RU` button in the title bar |

## Hotkeys

Default: `Ctrl + End`

Supported: `Ctrl`, `Alt`, `Shift`, `Win` + any key, `F1`–`F24`, `End`, `Home`, `Delete`, etc.

## Uninstall

```
NetSwitch.exe --uninstall
```

## Что делает

NetSwitch мгновенно отключает/включает все сетевые адаптеры по горячей клавише. Работает в системном трее, уведомления отображаются поверх полноэкранных игр.

## Установка

1. Скачайте [**NetSwitch.exe**](https://github.com/pro72rus-dev/NetSwitch/releases/latest/download/NetSwitch.exe)
2. Запустите файл — установка происходит автоматически
3. Готово! Нажмите `Ctrl + End` для отключения интернета

## Использование

| Действие | Как |
|----------|-----|
| Вкл/Выкл интернет | `Ctrl + End` |
| Открыть окно | Двойной клик по иконке в трее |
| Скрыть в трей | Кнопка `─` в заголовке |
| Сменить клавишу | **Change** → нажать комбинацию → **Confirm** |
| Сменить язык | Кнопка `EN`/`RU` в заголовке |

## Горячие клавиши

По умолчанию: `Ctrl + End`

Поддерживаются: `Ctrl`, `Alt`, `Shift`, `Win` + любая клавиша, `F1`–`F24`, `End`, `Home`, `Delete` и другие.

## Деинсталляция

